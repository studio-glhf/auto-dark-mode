# Privacy Policy

Auto Dark Mode Switch is local-only.

The extension checks visible page colors in the browser to decide whether a page appears light or dark. It saves only these local preferences in Chrome extension storage:

- whether the extension is globally enabled;
- the selected mode for a hostname, such as `Auto`, `Native`, `Force`, or `Off`;
- short-lived status information for the active tab.

The extension does not collect, transmit, sell, share, or remotely store browsing history, page content, personal information, analytics, identifiers, or settings.

The extension uses Chrome's `debugger` permission only to call Chrome DevTools Protocol commands that emulate native dark mode or Chrome Auto Dark Mode for the active tab.
